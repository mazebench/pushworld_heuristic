# Original Pushworld Level Files

There are 233 levels of Google Deepmind's pushworld. They sorted them into 4 categories of difficulty.

For some reason they are in .pwp format

We should translate them and place them all in the `benchmark` folder

Tiles are separated by spaces (inconsistent!) - translate them to single space

A is the player (translates to: “p.”)
AW is a gate: (translates to: )

M0 is mover 0 (translates to “a.”)
G0 is goal 0 (translates to “.a”)

G0+M1 means mover 1 stands on goal 0 (translates to “ba”)
